# 🚀 AI Trainer Assistant

An **AI-powered web application** that helps trainers and students generate structured learning content like notes, assignments, and explanations using Generative AI.

---

## 🌟 Features

* 💬 Ask AI anything (concept explanations)
* 📘 Generate **structured notes**
* 📝 Create **assignments automatically**
* ⚡ Fast responses using Gemini API
* 🎯 Beginner-friendly output format
* 🧠 Designed for trainers & students

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **AI Integration:** Google Gemini API
* **Environment:** dotenv

## ⚙️ Setup Instructions

### 🔹 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-trainer-assistant.git
cd ai-trainer-assistant
```

---

### 🔹 2. Install dependencies

```bash
cd backend
npm install
```

---

### 🔹 3. Add API Key

Create a `.env` file inside `backend/`

```
GEMINI_API_KEY=your_api_key_here
```

---

### 🔹 4. Run the backend

```bash
node server.js
```

Server will run at:

```
http://localhost:5000
```

---

### 🔹 5. Run frontend

Open:

```
frontend/index.html
```
